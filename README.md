# HyLight — Geotagged Photo Map

Fullstack web app for uploading geotagged aerial photos, displaying them on an interactive map, and generating AI descriptions via LLaVA.

## Stack

- **Frontend**: React + TypeScript + Vite + TailwindCSS + MapLibre GL
- **Backend**: Node.js + Express + PostgreSQL + PostGIS
- **Auth**: JWT (HTTP-only cookie)
- **Storage**: Cloudinary
- **AI**: Ollama + LLaVA (local, zero cost)
- **Deploy**: Docker Compose

## Prerequisites

- Docker + Docker Compose
- An [Ollama](https://ollama.com) instance with the `llava` model pulled
- A [Cloudinary](https://cloudinary.com) free account

## Setup

### 1. Clone the repo

```bash
git clone <repo-url>
cd hylight
```

### 2. Configure environment

```bash
cp .env.example .env
```

Edit `.env` with your Cloudinary credentials:

```env
CLOUDINARY_CLOUD_NAME=your_cloud_name
CLOUDINARY_API_KEY=your_api_key
CLOUDINARY_API_SECRET=your_api_secret
JWT_SECRET=any_random_secret
OLLAMA_URL=http://ollama:11434
```

### 3. Start

```bash
docker compose up
```

This starts 4 services: `web` (React, port 5173), `api` (Express, port 3000), `db` (PostgreSQL + PostGIS), `ollama` (LLaVA).

### 4. Pull the LLaVA model (first run only)

```bash
docker exec -it <project>-ollama-1 ollama pull llava
```

### 5. Open the app

Navigate to [http://localhost:5173](http://localhost:5173), register an account, and start uploading geotagged JPEGs.

## GPU acceleration (Linux + NVIDIA)

Install the [NVIDIA Container Toolkit](https://docs.nvidia.com/datacenter/cloud-native/container-toolkit/install-guide.html), then:

```bash
sudo nvidia-ctk runtime configure --runtime=docker
sudo systemctl restart docker
docker compose up
```

The `ollama` service will automatically use the GPU.

## Features

- Register / login with JWT auth
- Upload geotagged JPEG photos (GPS EXIF required)
- Interactive map with clustering (MapLibre GL + OSM tiles)
- Click a marker to view the photo, GPS coordinates, altitude, and capture date
- AI description auto-generated by LLaVA after each upload
